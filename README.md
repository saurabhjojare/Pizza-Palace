# Pizza Palace

## Overview

A comprehensive pizza ordering application designed to streamline the ordering process for pizza stores. This full-stack application allows users to browse the menu, customize their orders, and manage their cart. Admins can manage inventory, view orders, and update customer information.

## Features

- **Cart Management**: Add/remove pizzas, select size, add toppings, adjust quantities.
- **Customer Management**: Add, view, update, and delete customer information.
- **Order History**: View and track order history.
- **Business Rules**: Enforces validation rules for order customization and inventory.

## Technologies Used

- **Frontend**: ReactJS with TypeScript
- **Backend**: NestJS
- **Database**: PostgreSQL
- **API**: RESTful APIs
